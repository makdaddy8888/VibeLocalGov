What this is for

Many Australian council staff already have Microsoft 365 licences that include Copilot and Power Platform (Power Apps, Power Automate, Dataverse). The hard part is describing the problem so an assistant can help you design a responsible, reviewable app or flow.

This page collects your work context and biggest process pain, then gives you two text blocks: a thought process (why the prompt is built this way) and a natural-language prompt for Power Apps Vibe at vibe.powerapps.com. Microsoft’s docs describe that flow: you enter a description, agents work through a plan, data model, and app preview, then you save to a solution and publish when ready — see Create apps, data, and plans together using Power Apps vibe, Overview of the new Power Apps vibe experience, and Released versions of the new Power Apps experience (preview) for limitations (for example environment, Copilot, and region) that your admin can confirm. Do not paste resident names, addresses, or other personal information; describe processes and data types in generic terms.
